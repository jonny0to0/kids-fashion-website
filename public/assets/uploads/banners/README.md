# Hero Banner Images Directory

This directory stores uploaded hero banner images.

## Structure
- Desktop images: Uploaded with prefix `hero_desktop_`
- Mobile images: Uploaded with prefix `hero_mobile_`

## Permissions
- Directory permissions: 755
- File permissions: 644

## Supported Formats
- JPEG (.jpg, .jpeg)
- PNG (.png)
- WebP (.webp) - Recommended
- GIF (.gif)

## Recommended Image Sizes
- Desktop: 1920x600px
- Mobile: 768x400px

## Notes
- Images are automatically organized by the ImageUpload helper
- Old images are deleted when banners are updated or removed
- Maximum file size: 5MB per image
