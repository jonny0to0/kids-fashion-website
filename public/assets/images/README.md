# Images Directory

This directory contains all image assets for the Kids Bazaar e-commerce platform.

## Directory Structure

- `products/` - Product images
- `categories/` - Category images
- `banners/` - Homepage and promotional banners
- `avatars/` - User profile images

## Image Guidelines

- **Product Images**: Recommended size 800x800px, format: JPG/PNG/WebP
- **Category Images**: Recommended size 400x300px
- **Banners**: Recommended size 1920x600px
- **Avatars**: Recommended size 200x200px

## Adding Images

1. Upload product images through the admin panel (when implemented)
2. Or manually place images in appropriate directories
3. Ensure images are optimized for web (compressed)

## Placeholder Images

If no image is available, the system will use a placeholder. You can add a default placeholder image at:
`public/assets/images/placeholder.jpg`
